Welcome to ClassMaker!

ClassMaker is a library for creating classes at runtime. Taking dynamic EPackage as an input, it invokes the generator (currently Ecore default), then PDE exporter, then installs the result into host and supplies EPackage class, loaded into its own OSGi runtime, to client code. Here is a Javadoc.

The purpose of it is, for instance, applying an adaptive object model architecture. Further work is follows.
